CONTEXT : Holweg is a world leader in producing paper bag-formation machines. However, due to poorly optimized machine parameters, paper bands often tear leading to waste.

Predictive maintenance and Anomaly detection of machining tools during milling process through vibration analysis
âž¤ Academic last year project (2nd year Master's degree)

CONTEXT : During the milling process, high energy shocks appear between the machined piece and the tool, producing a distinct vibration. Sometimes, the tool gets damaged during this process. This damage reduces the machine's lifespan as the vibration gets transmitted in unwanted areas. Moreover, predicting the Remaining Useful Life of tools (RUL) is complicated because it needs special tools and skills.

MY ROLE : Through vibration datas, design, train and implement an AI model capable of detecting different class anomalies, and predict the remaining useful life of tools.

DURATION : 3 months (sep 2024 - feb 2025)

- Designed, trained and tested different AI model through our State Of the Art study : CVAE, Transformers, CNN pretrained models, custom LSTM-AE

- Used Python for primary coding language and Pytorch as our main library for quicker and more robust implementation

- Overcame lack of data through heavy signal and data preprocessing

- Demonstrated log-spectrogram images are superior to time-series data for precision

TOOLS : Python (Pytorch, VSCode), Microsoft Excel

RESULTS : > 97 % accuracy on the anomaly detection model
< 3 % error on predicting the RUL

AMR Intelligent object avoidance system - ROS
âž¤ Academic project (1st year Master's degree)

CONTEXT : To introduce us to industrial constraints, a first project was organized for our Master's degree. This subject was inspired by Amazon's Autonomous Mobile Robots (AMR) capable of taking a package and delivering it from point A to point B while avoiding obstacles. 

OUR ROLE : Use the existing TurtleBot3 robot to implement our own object avoidance system through Robot Operating System (ROS)

DURATION : 6 months (oct 2023 - apr 2024)

- Introduced ROS for robust and easy real-time application

- Implemented different algorithms for performance evaluation (A*, APF, Djisktra)

- Restrict ourselves to lightweight models for our small embedded system (Raspberry PI)

- Firmware and Software control for servomotor controls, Raspberry PI and Arduino communication

TOOLS : ROS 2, Ubuntu 22.04, Python, C++, Nav2, VSCode, Arduino IDE

RESULTS : Failed to implement a reinforcement learning AI algorithm
Instead, we used a method called Artifical Potential Field (APF) to simulate forces for obstacles and end-point goal

Cybertank
âž¤ Academic Open project (3rd year Bachelor's degree)

CONTEXT : For our final Bachelor's year, the Faculty organized an Open project that allowed students to propose and execute their own custom project. Our project was inspired by Tesla's Cybertruck : a robust and mobile vehicle. Thus, our vehicle named "CyberTank" was given Cybertruck's bodywork but with tracks for added "coolness".

OUR ROLE : Create our CyberTank from scratch using available tool and minimal spending

DURATION : 4 months (jan 2023 - may 2023)

- Electrical sizing of all components (battery, Arduino, motor driver, servo motor, camera)

- Use of a 3D-printed gearbox: a double differential inspired by the transmission systems of vintage tanks

- Mechanical dimensioning of printable parts using Inventor: custom suspension, double differential, and bodywork

TOOLS : VSCode, Arduino IDE, Inventor, 3D printer

RESULTS : Double differential ended up breaking because of the high torque
